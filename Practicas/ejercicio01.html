📘 Ejercicio de Examen 1: Gestión de Biblioteca (LocalStorage y JSON)

Objetivo: Trabajar con clases/objetos, manipulación de arrays, conversión de JSON y Web Storage.

Descripción: Desarrolla una pequeña aplicación para gestionar los libros de una biblioteca.

1. Crea una clase Libro con las propiedades: id, titulo, autor, anioPublicacion y leido (booleano).
2. Simula una base de datos inicial creando un String en formato JSON que contenga un array con 3 libros.
3. Transforma ese JSON a objetos de JavaScript utilizando JSON.parse().
4. Utiliza un bucle para recorrer ese array y guarda cada libro individualmente en el LocalStorage utilizando su id como clave y el objeto libro convertido a JSON como valor.
5. Calcula cuántos libros han sido leídos y guarda ese número en el SessionStorage con la clave libros_leidos.
6. Añade una funcionalidad donde se busque (usando el método .find()) un libro por su título y se imprima su información por consola.
