📘 Ejercicio de Examen 2: Control de Gastos Personales (DOM básico, Eventos y Arrays)

Objetivo: Capturar eventos del HTML, leer valores de inputs, instanciar objetos y guardar históricos.

Descripción: Crea una interfaz HTML muy básica con 3 inputs: "Concepto" (texto), "Cantidad" (número) y "Tipo" (ingreso o gasto). Añade un botón "Registrar".

1. En tu archivo .js, crea un array global vacío llamado historialMovimientos.
2. Crea una función que se ejecute en el evento onclick del botón.
3. Dentro de la función, utiliza document.getElementById('id').value para recoger los datos de los 3 inputs.
4. Crea un objeto literal o instanciado con esos datos y añádelo al array historialMovimientos usando .push().
5. Utiliza una estructura iterativa (como for...of) para recorrer el array y calcular el saldo total (sumando si es ingreso, restando si es gasto).
6. Imprime el saldo total en una etiqueta

del HTML utilizando .innerText.
