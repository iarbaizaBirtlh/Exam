📘 Ejercicio de Examen 3: Sistema de Votaciones (Bucles, Condicionales y Switch)

Objetivo: Control de flujo avanzado y lógica de programación.

Descripción: Desarrolla un script para contabilizar los votos de unas elecciones con 3 candidatos (A, B y C).

1. Crea un array vacío llamado votos.
2. Simula la entrada de votos generando 20 números aleatorios entre 1 y 4 (usando Math.random()).

    1 = Voto para A
    2 = Voto para B
    3 = Voto para C
    4 = Voto nulo

3. Usa un bucle for para rellenar el array votos con estos números generados.
4. Recorre el array de votos y utiliza una estructura switch para ir sumando los votos en variables contadoras independientes (votosA, votosB, votosC, nulos).
5. Al final, utiliza condicionales if-else para determinar qué candidato tiene más votos e imprime por consola el ganador y el recuento total.
